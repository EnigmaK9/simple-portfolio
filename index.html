  • Pathfinding

    Programa que resuelva el problema de pathfinding para un mapa de 20x20 celdas con al menos 40 celdas obstáculo. El programa recibe como entrada la posición inicial de dos agentes A1 y A2, y encuentra el camino mínimo para llegar cada uno a su destino. El algoritmo de pathfinding es Dijkstra. Se implementa usando Python y la biblioteca de pygame.
